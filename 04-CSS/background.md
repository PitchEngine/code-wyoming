## Default
All elements have a default transparent background

### Color
Try to always use a color, on the offchance your image can't load.

* rgb
* hex
* hsla
* color
* name

## Background-Image
url("")

* optional spaces
* optionally no, single, or dbl quotes
* on failure, goes to "none"

## Background-repeat

* repeat
* no-repeat
* repeat-x
* repeat-y
