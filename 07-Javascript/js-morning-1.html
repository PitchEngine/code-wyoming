Data types

Numbers
  • 12
  • 3e-9
  • 3*5
  • 5%2 //=> 1
  • 43/0 //=> Infinity
  • Only so precise...
Strings
A stringof characters
Surrounded by either "or '.
Use \to escapequotes, ie 'Here\'s an example.' //=> "Here's an example"
"Concatenate" strings using +, ie
"Hello,"+" world." //=> "Hello, world./"
Booleans
true or false

Comparison Operators

The following can all be used to turn strings, numbers into booleans.
  • == (equal to)
  • === (strictly equal to)
  • != (not equal to)
  • !== (strictly not equal to)
  • > (greater than)
  • < (less than)
  • >=
  • <=

Built In Functions

alert
Takes a string, pops up a box for the user to hit OK on
confirm
Takes a string, pops up a box with Yes/No.
Returns true/false depending on user input.
prompt
Takes a string. Pops up a text box for the user to type input into, returns whatever the user typed.
console.log
Prints its input to the console
